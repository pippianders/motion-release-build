# motion-release-build

Docker script to build multiple motion releases

# Supported platforms

* ubuntu:xenial
* ubuntu:bionic
* ubuntu:focal
* debian:stretch
* debian:buster

# Supported archinectures
 
 * amd64
 * i386
 * arm64v8
 * arm32v7
  
 # Not supported
 
 * i386/ubuntu:focal - libmariadbclient-dev not available
