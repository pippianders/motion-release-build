# motion-release-build

Docker script to build multiple motion releases

# Supported platforms

## amd64

* ubuntu:xenial
* ubuntu:bionic
* ubuntu:focal
* debian:stretch
* debian:buster

## i386
 
 * i386/ubuntu:xenial
 * i386/ubuntu:bionic
 * i386/debian:stretch
 * i386/debian:buster
  
 # Not supported
 
 * i386/ubuntu:focal - libmariadbclient-dev not available
